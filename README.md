# libargtable3

Stewart Heitmann's Argtable3 command-line argument parsing library packaged up
for local use (as a git submodule) with a Visual Studio solution for building on
Windows.

This repository is __not__ affiliated with either Stewart Heitmann, nor with the
current Argtable3 maintainer Tom G. Huang.

The current version of Argtable3 may be found at it's official repository:

* [Argtable3](https://github.com/argtable/argtable3)

Please support the official release.
